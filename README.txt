terminalClipboard (cb)

A basic windows-style clipboard for linux terminal
Jordan "Jorb" Dehmel, 2022-23, jdehmel@outlook.com

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Installation:

    In this directory, enter the command `make install`.
You can now use the command `cb <COMMAND>` to call this
program. See the commands section below for a list of all
commands.

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Commands:

help - Print a list of all commands

copy <FILENAME> - Copy a file
cut <FILENAME> - Cut a file
paste - Paste a file to the current directory

switch <NAME> - Switches to the indicated clipboard
list - Lists all currently existing clipboards
clear <NAME> - Empty a clipboard
remove <NAME> - Delete a clipboard
